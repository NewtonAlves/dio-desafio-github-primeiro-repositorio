#GIT

- Links -

Link do site Git: https://git-scm.com/
Link para download do Git: https://git-scm.com/downloads

- Baixando o Git -

escolha o git de acordo com seu sistema operacional, onde no meu caso é windows mas no site tem opções para windows, linux e mac.
escolhi git windows x64
executo após o download terminar

- Instalação Git -

Leia os termos de uso e selecione Next
Selecione ou use o caminho padrão e Next
Escolha as opções que mais seja de seu uso e Next
Se deseja alternar nome do caminho de grupo start, deixei padrão e Next
Selecione um editor de texto de sua preferencia ou use padrão e Next
Ajustando o nome da branch inicial de repositorios, use o padrão temporariamente e pode ser mudado após a instalação e Next
Ajuste o seu caminho ou use padrão e Next
As demais escolhas usei padrão como uso da propria Git e finalizei aos demais Next até finalizar instalação.

ao finalizar o Git GUI e Git Bash estarão de facil acesso no contexto do menu ao clicar segundo botão do mouse ou touchpad.

O Git Bash é um terminal extendido para otimizar o uso do Git.

#GITHUB

- Links -

Link do site GitHub: https://github.com/
Link para criar conta no GitHub: https://github.com/signup

- Cadastro GitHub -

digite seu email
Digite sua senha padrão para logar (não esqueça dela, pós tanto email quanto a senha são necessarias para entrar na sua conta)
Digite o nome para conta, exemplo: MinhaContaPadrao
Efetue a verificação de segurança
Escolha seu plano de conta
Escolha as preferências
Concluindo cadastro ao preencher todas as informações necessárias e preferências.
Confirme seu endereço de email

- Criando Repositorio - 

clique na sua foto de perfil no canto superior direito da página inicial do GitHub e selecione a opção "Your repositories" no menu suspenso.

No canto superior direito da página "Your repositories", você verá um botão verde com a inscrição "New" clique nele.
Você será direcionado para a página de criação de um novo repositório.
Preencha as informações do repositório:

Escolha um nome para o seu repositório. O nome deve ser único na sua conta do GitHub.
Você pode optar por tornar o repositório público ou privado.
Lembre-se de que os repositórios privados geralmente requerem uma assinatura paga.
Adicione uma descrição para o seu repositório, se desejar.
Escolha se deseja inicializar o repositório com um arquivo README.
Isso é útil para fornecer informações sobre o projeto.
Escolha as configurações adicionais (opcional).

Você pode adicionar um arquivo .gitignore para especificar quais arquivos e diretórios devem ser ignorados pelo Git.
Você também pode adicionar uma licença ao seu projeto, o que é uma boa prática para especificar como outras pessoas podem usar seu código.
Clique em "Create repository" (Criar repositório): Após preencher todas as informações necessárias, clique no botão verde "Create repository" (Criar repositório) na parte inferior da página.
